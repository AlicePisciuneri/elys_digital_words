# elys-portfolio

Un portfolio personale in costruzione, pensato per raccogliere in un unico spazio il mio percorso nel digitale e il mio lavoro di redattrice.

Questo progetto nasce con un doppio obiettivo: creare una presenza online che mi rappresenti davvero e riprendere mano con il codice attraverso una struttura costruita passo dopo passo.

## Cosa conterrà

- una sezione dedicata al profilo digitale e alle competenze web
- una sezione dedicata al lavoro editoriale e ai contenuti scritti
- link ai progetti, ai contatti e ai profili professionali
- una panoramica di ciò che posso offrire
- un piccolo diario di bordo per raccontare l’evoluzione del progetto

## Tecnologie previste

- HTML
- CSS
- React
- Node.js
- Express

## Stato del progetto

Il progetto è attualmente in fase di progettazione e costruzione dello scheletro iniziale.
